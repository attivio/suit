#### Examples:

All the examples use the same date, with different formatting options.

__1.__ ISO_8601

```jsx
  <FormattedDate
    date={new Date('2015-10-21T21:29:00-00:00')}
    format={FormattedDate.DateFormat.ISO_8601}
  />
```
